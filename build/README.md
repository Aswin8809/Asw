# Simple Interest Calculator

## Description
This project is a simple interest calculator that calculates the interest based on user input.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Inputs
- Principal amount (P)
- Rate of interest (R)
- Time period (T)

## Output
- Simple Interest

## How it works
The user enters the principal amount, rate of interest, and time period.  
The calculator then applies the formula and displays the simple interest.

## Example
If:
Principal = 1000  
Rate = 5%  
Time = 2 years  

Simple Interest = (1000 × 5 × 2) / 100 = 100
